3. The function will return the sum of the two numbers.

## Code

```
const addTwoNumbers = (a, b) => {
  const sum = a + b;
  const product = a * b;

  console.log(`The sum of ${a} and ${b} is ${sum}`);
  console.log(`The product of ${a} and ${b} is ${product}`);
};
```
